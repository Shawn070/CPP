## 6.2.4 指针的赋值
1. 可以声明指向常量的指针，此时不能通过指针来改变所指对象的值，但指针本身可以改变，可以指向另外的对象。例如:
``` c++
int a;
const int * p1 = &a;
int b;
p1 = &b;
*p1 = 1;	// 错误
```
2. 可以声明指针类型的常量，此时指针本身的值不能被改变。例如:
``` c++
int * const p2 = &a;
p2=&b;		// 错误
```
3. 任何类型的指针都可以赋值给void类型的指针变量。经过使用类型显示转换，通过void类型的指针便可以访问任何类型的数据。
